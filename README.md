# mongoose-express-02

## Development Workflow

Run in Development Mode

```bash
  npm run start:dev
```

Run in Production Mode:

```bash
  npm run start:prod
```

Build Project:

```bash
  npm run build
```

## API Reference

#### Domain: https://assingment2.vercel.app/

#### Get all users

```http
  GET /api/users
```

#### Get single user

```http
  GET /api/users/:id
```

#### create user

```http
  POST /api/users
```

#### Update user by id

```http
  PUT /api/users/:id
```

#### Delete user by id

`http
  Delete /api/users/:id
`
